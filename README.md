# Siflow 标注写入飞书表格工具

## 用法

1. 安装 Tampermonkey，并在 Chrome 扩展详情里开启“允许用户脚本”。
2. 新建油猴脚本，把 `siflow_tampermonkey.user.js` 的内容完整复制进去并保存。
3. 双击 `一键启动-正式写入.command`。
4. 浏览器会打开 `Siflow 写表控制台`。
5. 在控制台里填写“下一次写入行号”，点击“跳到此行号”。不填则自动找表格下一空行。
6. 打开 Siflow 标注详情页，右下角会出现 `F8 写入表格`。
7. 每个 case 按一次 `F8`，写入成功后控制台行号会自动加 1。

## 写入列映射

目标表头：

```text
prompt | A | 打分 | 理由 | B | 打分 | 理由 | C | 打分 | 理由 | D | E
```

写入内容：

```text
prompt -> A列
A      -> B列
B      -> E列
C      -> H列
D      -> K列
```

中间的 `打分/理由/E` 会留空。

## 注意

- 本工具默认允许覆盖目标行，适合手动指定行号后重写。
- `585字 / 1307字 / 180字` 这类页面字数统计会被自动清理。
- 油猴脚本匹配 `https://*.siflow.cn/*`，只在页面存在“主内容区”卡片时显示按钮。
- `.siflow_service_state.json` 和 `.siflow_written_log.jsonl` 是本地状态/日志，已被 `.gitignore` 忽略，不要提交。
